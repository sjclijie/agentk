function listen(port, cb)

Create a new http server, bind it to a port or socket file. A callback is supplied which accepts a http request object as parameter and returns a HttpResponse

Example

   http.listen(8080, function(req) {
     return http_response.ok()
   });

Parameters

port number|string
TCP port number or unix domain socket path to listen to
cb function|router::Router
request handler callback

Returns

node.http::Server
