engines
constant

engines:object

map of view engines, user can supply a specific view engine by assigning to this object

Example

 view.engines.jade = require('jade').__express;
path
variable

path:string = ''

directory of view template files (default to current directory)

view_engine
variable

view_engine:string = 'ejs'

default view engine when no extension name is supplied

module_loader
variable

module_loader:function

method used to load view engine by extension, default to require. User can supply a specific loader by assigning this variable

Example

 view.module_loader = function(name) {
     return require(name).__express
 }
render()
method

function render(name, locals, mimeType)

render a template file into response content, returns a HttpResponse. User should specify content type if needed.

Parameters

name string

template name, with or without extension

locals object

local bindings

mimeType string

custom mimeType, default to 'text/html'

Returns

HttpResponse
